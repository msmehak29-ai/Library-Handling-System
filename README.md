# Library-Handling-System
Simple Python based Library Handling System
Overview

A simple yet efficient Library Handling System built using Python to manage books, members, and operations in library. It helps library staff efficiently manage book inventory , borrowing / returning operations by maintaining records of all transactions and providing basic reporting capabilities.

Features

•	Book Management: Add, update, delete, and search books in the library inventory
•	Member Management: Register new members and manage member information
•	Borrowing System: Handle book lending and returning operations
•	Transaction History: Maintain records of all borrowing and returning activities
•	Reporting: Generate basic reports on available books, borrowed items, and member activity

Technologies/Tools Used

•	Programming Language: Python 3.x

Prerequisites

•	Python 3.6 or higher installed on your system
•	Basic knowledge of running Python scripts
